Functions :
preload: preload game assets such as images, audio files, spritesheets, and JSON data
create: initialize game objects, set up the game world, and perform any other setup tasks needed for the game to start. You can create sprites, add physics, set up input handling 
update: update the game state, handle user input, perform collision detection, and implement game logic. Inside update(), you can check for input from the player, move game objects, detect collisions, and update game variables
render: called once per frame, immediately after update() completes.

update update le gamestate, et render permet de réagir aux changement (update : colision? > render : sprites de destruction | udate : powerup > render : score update)
